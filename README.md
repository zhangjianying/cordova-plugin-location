# cordova-plugin-location
获取坐标位置


# 安装
```javascript
cordova plugin add https://github.com/zhangjianying/cordova-plugin-location.git
```



# 使用

```javascript
 TWLocation && TWLocation.getLocation(onSuccess,onFail);

==> {
        code:0,
        desc:'web端测试坐标',
        latitude:26.57,
        longitude:106.72,
        provider:'GPS'
      }
```
